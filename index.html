01 · The Project

A final project with flavor.

Parmco is the Group 8 capstone deliverable for ECSE 4235 — an end-to-end engineering project taking an idea from spec to working system. The build, the documentation, the testing, the demo: all of it lives in our repository, and all of it is the work of four engineers who decided that "boring nameplate" was not on the menu.

This site is the public face of that work. The code, schematics, write-ups, and revision history all live one click away on GitHub.

02 · What's Inside

Built in three layers.

/ 01

Hardware

The physical layer — boards, sensors, actuators, and the wiring that makes them all play nicely together. Designed, assembled, and bench-tested by the team.

/ 02

Firmware

The brain. Embedded code handling I/O, timing, control loops, and the ugly edge cases that don't show up until 2 a.m. the night before the demo.

/ 03

Documentation

Schematics, BOMs, test results, and a README that actually explains how to run the thing. Written so future students (and future us) can pick it up cold.
